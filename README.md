# Fabled Compass

**Every career has a story.**

Fabled Compass is a career platform that combines a job board with interactive workplace roleplay scenarios. Job seekers can practice real-world situations, receive structured feedback, earn skill badges, and present demonstrated abilities to employers alongside their traditional experience.

## Product Pillars

1. **Careers** — Find and apply for employment opportunities.
2. **Adventures** — Complete roleplay-based workplace scenarios.
3. **Community** — Connect through professional and social roleplay features after the core platform is established.

## MVP Goal

The first release must allow a user to:

1. Create an account.
2. Select a job-seeker or employer profile.
3. Browse job listings.
4. Open and complete one branching workplace scenario.
5. Receive structured feedback.
6. Earn a badge displayed on their profile.

The first scenario is **The Difficult Customer**, which awards the **Customer Service Foundations** badge when its requirements are met.

## Initial Technology Direction

- Next.js
- TypeScript
- Tailwind CSS
- Supabase
- Vercel

AI-generated conversations are not part of the first scenario engine. The MVP will use authored branches and transparent scoring rules before AI features are introduced.
