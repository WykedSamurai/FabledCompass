# Fabled Compass MVP Completion Plan

## Phase 1 — Scenario Badge System

- Give each scenario one distinct badge identity.
- Store badge definitions separately from user awards.
- Show scenario, skills, score, version, and date in badge details.
- Replace the generic Customer Service Foundations badge with the scenario-specific Customer Recovery badge.

## Phase 2 — Profile Structure and Privacy

- Convert experience, education, certifications, and skills into structured entries.
- Separate private account data from employer-visible profile data.
- Improve resume import review and field-mapping controls.

## Phase 3 — Job-Seeker Workflow

- Replace sample jobs with database-backed listings.
- Add saved jobs and applications.
- Add application status history.

## Phase 4 — Employer Workflow

- Add employer profiles.
- Add job creation, editing, publishing, and closing.
- Add candidate and application review.
- Show relevant scenario evidence and badges to employers.

## Phase 5 — Account and Operational Safeguards

- Password reset.
- Account deletion.
- Profile visibility and privacy controls.
- Upload size and file validation.
- Error, loading, and empty-state cleanup.
- Mobile and accessibility testing.

## Completion Rule

Do not begin social features, advanced community systems, or additional roleplay expansion until Phases 1 through 5 are complete and tested.
