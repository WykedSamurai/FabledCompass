# Project Signal

## Purpose

Project Signal defines and builds the Communication Hub for Fabled Compass.

The Communication Hub is a permanent communication layer, not a single chat page.

## Philosophy

The main workspace is where users work.

The Communication Hub is where conversations happen.

It should feel like mission control: calm, organized, contextual, and purposeful.

## Conversation Types

The same conversation engine should eventually support:

- Compass AI
- Direct messages
- Friend chats
- Group and guild chats
- Recruiter conversations
- Employer conversations
- Notifications
- Out-of-character adventure chat
- Shared platform objects

## Compass AI

Compass is a pinned conversation and a contextual coach.

Compass can help users:

- explain scores
- improve portfolio content
- review resumes
- understand job match
- prepare for interviews
- receive scenario guidance
- manage roleplay OOC questions

## Context

Every conversation should understand what it belongs to.

Examples:

- Friend conversation
- Application conversation
- Employer conversation
- Scenario OOC conversation
- Guild conversation
- Compass conversation on Portfolio

## Shared Objects

Messages should eventually support platform objects, not only plain text.

Examples:

- Resume
- Portfolio
- Scenario Report
- Competency Evidence
- Job Posting
- Assessment Result
- Interview Feedback

## Early Version

The first version may include:

- floating compass button
- right-side drawer
- pinned Compass placeholder
- notifications placeholder
- OOC placeholder for future scenarios

It should be built as an architecture foundation, not a throwaway chat widget.
