# Atlas Component Library

Atlas components are the shared visual foundation for Fabled Compass.

Rule: if a UI element appears more than once, it should become an Atlas component.

Current primitives:

- `AtlasSky` — global living sky layer
- `AtlasCard` — reusable glass card
- `AtlasPanel` — larger glass surface
- `AtlasSection` — layout grouping
- `AtlasHeader` — shared eyebrow/title/description header
- `AtlasDrawer` — drawer surface
- `AtlasDialog` — dialog surface
